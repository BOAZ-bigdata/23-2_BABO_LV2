### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    냅색(Napsack)알고리즘!!!
    
### 코드의 시간복잡도와 그 이유:
    O(NxK)


### 코드:
```python
import sys
input = sys.stdin.readline
 
n,k = map(int,input().split())

items = []
for i in range(n):
    w,v = map(int,input().split())
    items.append((w,v))
    print(items)
dp = [0 for _ in range(k+1)]
for item in items:
    w,v = item
    for i in range(k,w-1,-1):
        dp[i] = max(dp[i],dp[i-w]+v)
print(dp[-1])
```
